# Nomachine

## Install

```bash
# 主机和客户端都需要下载该工具
$ wget -c https://download.nomachine.com/download/8.11/Linux/nomachine_8.11.3_4_amd64.deb
$ sudo dpkg -i nomachine_8.11.3_4_amd64.deb
```
